# ROBOTS

This repository is intended for AI-assisted development and evaluation of the DataDivas assignment app.

Guidelines for automated agents:
- Use the Python standard library only.
- Do not add non-standard dependencies.
- Preserve the desktop GUI interface in `main.py` / `datadivas/gui.py`.
- Keep the assignment logic in `datadivas/assignment.py`.
- Do not modify `README.md` except to document features, installation, or usage.
- Do not expose any personal or private data.
- If provided sample data is altered, preserve the same input format.
